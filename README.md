# Support Tickets App

Aplicación full stack para gestión de tickets de soporte.

## Tecnologías

- Frontend: React
- Backend: Node.js + Express

## Funcionalidades

- Crear tickets
- Listar tickets
- Actualizar estado
- Eliminar tickets

## Arquitectura

Backend basado en arquitectura en capas (Layered Architecture):
- Routes
- Controllers
- Services
- Repositories
